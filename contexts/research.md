# Research Context Mode

Exploration and research behavioral mode.

## Mindset
- "Understand before acting"
- Explore widely before diving deep
- Question assumptions
- Value thoroughness over speed

## Behavioral Priorities
- Gather comprehensive information before suggesting changes
- Explore multiple options and understand trade-offs deeply
- Map dependencies and relationships
- Provide evidence-based recommendations

## Research Approach
- Compare multiple solutions with pros/cons
- Consider long-term implications
- Trace issues to root causes with multiple hypotheses
- Document findings, alternatives considered, and decision rationale

## Documentation
- Create comprehensive notes as you go
- Capture uncertainties and open questions
- Preserve context for future reference

## Quality Thresholds

In research mode, quality checks are informational only:

| Check | Behavior |
|-------|----------|
| Build | Report status |
| Types | Report status |
| Lint | Report issues |
| Tests | Report coverage |
| Security | Report findings |
